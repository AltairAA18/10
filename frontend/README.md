# 🎨 Frontend - Blog

Фронтенд приложение для блога на чистом JavaScript, HTML и CSS.

## ✨ Возможности

- 📝 Просмотр всех опубликованных статей
- 🏷️ Фильтрация по категориям
- ❤️ Лайкирование статей
- 👁️ Счетчик просмотров
- 👤 Регистрация и авторизация пользователей
- ✍️ Создание собственных статей
- ✏️ Редактирование своих статей
- 🗑️ Удаление своих статей
- 👨‍💼 Просмотр профиля с личными статьями
- 📱 Адаптивный дизайн

## 🚀 Быстрый старт

### Способ 1: Примерный Python HTTP Server

```bash
cd frontend

# Python 3
python -m http.server 8000

# или Python 2
python -m SimpleHTTPServer 8000
```

Откройте браузер: **http://localhost:8000**

### Способ 2: Node.js HTTP Server

```bash
cd frontend

# Установить http-server (один раз)
npm install -g http-server

# Запустить
http-server
```

Откройте браузер: **http://localhost:8080**

### Способ 3: Live Server в VS Code

1. Установите расширение "Live Server"
2. Клик правой кнопкой на `index.html`
3. Выберите "Open with Live Server"

## 📚 Как использовать

### Просмотр статей

1. Откройте приложение
2. На главной странице отображаются все опубликованные статьи
3. Кликните на карточку, чтобы прочитать полную статью

### Фильтрация по категориям

Используйте кнопки фильтра:
- **Все** - показать все статьи
- **Технология** - только статьи о технологиях
- **Жизнь** - статьи о жизни
- И т.д.

### Регистрация

1. Нажмите **Регистрация** в навигации (в правом верхнем углу)
2. Заполните форму (имя, email, пароль)
3. Нажмите **Зарегистрироваться**

### Вход

1. Нажмите **Вход** в навигации
2. Введите email и пароль
3. Нажмите **Войти**

### Написать статью

1. Авторизуйтесь
2. Нажмите **✍️ Написать** в навигации
3. Заполните форму:
   - **Название** - название статьи
   - **Выдержка** - краткое описание (опционально)
   - **Содержание** - основной текст статьи
   - **Категория** - выберите категорию
   - **Теги** - добавьте теги через запятую (опционально)
   - **Опубликовать** - отметьте, если готово к публикации
4. Нажмите **Опубликовать**

### Редактирование статьи

1. Откройте вашу статью (на странице профиля или из списка)
2. Нажмите **✏️ Редактировать** (доступно только для ваших статей)
3. Отредактируйте необходимые поля
4. Нажмите **Сохранить**

### Профиль пользователя

1. Авторизуйтесь
2. Нажмите **👤 Профиль** в навигации
3. Посмотрите список ваших статей
4. Отредактируйте или удалите любую статью

## 🐛 Troubleshooting

### "CORS error"
**Проблема**: Не удается подключиться к API

**Решение**:
1. Убедитесь, что backend сервер запущен на `http://localhost:5000`
2. Проверьте URL в `js/api.js`
3. Убедитесь, что CORS включен на backend

### "Failed to load articles"
**Проблема**: Статьи не загружаются

**Решение**:
1. Проверьте консоль браузера (F12)
2. Убедитесь, что MongoDB подключена
3. Проверьте, что есть опубликованные статьи

### "Token expired"
**Проблема**: Токен истек

**Решение**:
1. Выполните выход (нажмите Выход)
2. Войдите заново

## 🚀 Развертывание

### На GitHub Pages

1. Создайте GitHub репозиторий
2. Загрузите папку `frontend` в ветку `gh-pages`
3. Включите GitHub Pages в настройках
4. Приложение будет доступно по адресу: `https://username.github.io/repository/frontend/`

**Важно**: Измените `API_URL` на ваш production сервер

### На собственном сервере

1. Загрузите папку `frontend` на веб-сервер
2. Настройте веб-сервер для обслуживания статических файлов
3. Измените `API_URL` в `js/api.js`
4. Настройте CORS на backend

3. Очистите кэш браузера (Ctrl+Shift+Del)
4. Перезагрузите страницу (Ctrl+R или Cmd+R)
